# 🖼️ Quick Image Upload Guide

## Where to Put Your Images

All images go in: **`/public/images/`**

## What Images to Upload

### 1. Profile Photo
- **Filename**: `profile.png`
- **Used**: Hero section (your portrait)
- **Size**: 800 x 1200px (portrait)
- **Format**: PNG or JPG

### 2. Portfolio Images
- **Filenames**: `project1.png`, `project2.png`, ... `project7.png`
- **Used**: Featured Work - Graphic Design Portfolio
- **Size**: 1200 x 900px (landscape)
- **Format**: PNG or JPG

## How to Upload

1. Go to `/public/images/` folder
2. Replace the SVG placeholder files with your actual images
3. **Keep the exact same filenames**
4. Done! No code changes needed.

## Important Rules

✅ **DO**:
- Use the exact filenames listed above
- Optimize images for web (< 500KB each)
- Use PNG or JPG format

❌ **DON'T**:
- Rename the files
- Change image paths in code
- Use different formats (like WebP or AVIF)

## Deployed Without Real Images?

No problem! The site has placeholder SVG images that:
- Show where each image goes
- Match your site's design
- Work perfectly until you upload real images
