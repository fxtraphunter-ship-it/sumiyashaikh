## Image Format Guidelines

- **Format**: PNG or JPG
- **Quality**: High resolution for web (optimize before upload)
- **Naming**: Use lowercase, descriptive names with hyphens
- **Size**: Optimize images to keep file sizes reasonable (< 500KB per image)

## Upload Instructions

1. Place your images in this `/public/images` directory
2. Ensure filenames match exactly as listed above
3. Images will be automatically served at `/images/[filename]`
4. No build step required - images are served statically
