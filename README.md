# Sumiya Shaikh Portfolio

A premium modern personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- 🎨 Dark theme with glassmorphism UI
- ✨ Gen-Z aesthetic with purple/blue gradients
- 🎭 Smooth animations with Motion (Framer Motion)
- 📱 Fully responsive design
- 🚀 Optimized for production
- 🎪 Interactive carousel for work showcase
- 📧 WhatsApp integration for contact

## Tech Stack

- **Framework:** React 18.3
- **Build Tool:** Vite 6.3
- **Styling:** Tailwind CSS v4
- **Animations:** Motion (Framer Motion)
- **Routing:** React Router 7
- **Icons:** Lucide React
- **Carousel:** React Slick
- **TypeScript:** 5.5

## Getting Started

### Prerequisites

- Node.js 18+ or higher
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Import project in Vercel
3. Vercel will auto-detect Vite and use these settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
   - **Install Command:** `npm install`

Or use the Vercel CLI:

```bash
npm install -g vercel
vercel
```

### Deploy to Netlify

```bash
# Build the project
npm run build

# Deploy dist folder to Netlify
```
